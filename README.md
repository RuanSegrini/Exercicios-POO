# Exercícios de POO em Java

Este repositório contém **exercícios diários de Programação Orientada a Objetos (POO) em Java**, focando em:

- Classes, objetos, herança e polimorfismo
- Encapsulamento e interfaces
- Generics, Set e Map
- Lambda, Method Reference e Stream API
- Uso de LocalDate e LocalDateTime
- Organização em camadas (model, repository, main)

Cada exercício é uma **mini-aplicação ou desafio**, com regras específicas e implementações próprias, pensado para **praticar conceitos de Java avançado** e **preparação para mercado/portfólio**.

---

## Como usar

1. Navegue até a pasta do exercício que quer executar.
2. Abra o projeto no IntelliJ ou outro IDE Java.
3. Compile e rode o `Main.java` de cada exercício.
4. Cada pasta contém um **README.md próprio**, explicando o objetivo, regras e instruções de execução.

---

## Objetivo

- Praticar **POO completo** em Java
- Criar mini-projetos **organizados em camadas**
- Aprender a usar **coleções, streams, lambda e datas**
- Construir um **portfólio funcional** de exercícios para GitHub
